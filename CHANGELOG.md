# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of WECS
- Core ECS architecture (Entity, Component, System, Query, World)
- Fine-grained reactive system (Signal, Computed, Effect)
- Client-side rendering with Virtual DOM
- Server-side rendering (SSR) with hydration
- RPC system for client-server communication
- State synchronization with delta compression
- Isomorphic component definitions
- Comprehensive test suite
- Build system with esbuild
- GitHub Actions CI/CD pipeline
- Todo app example demonstrating all features

## [0.1.0] - 2025-11-24

### Added
- Initial public release
- Full TypeScript support
- Zero runtime dependencies
- Tree-shakeable exports
- ESM module format
- Comprehensive documentation
